# **Найди пару | Memory Game**

*Игра на память ддля тренировки внимания: пользователь открывает карты по две, находит одинаковые пары и зарабатывает очки. Проект реализован на чистом HTML,CSS,JS с хранением коллекций карт в JSON и адаптивным дизайном*

## Использование
После открытия игры вы увидите игровое поле с перевернутыми картами

**Правила игры**

1. Кликните на любую карту, чтобы перевернуть ее
2. Затем кликните на вторую карту
3. Если картинки совпали - пара остается открытой, вы получаете +10 очков
4. Если не совпали - карты переворачиваются обратно
5. Игра засчитывает количество попыток
6. Победа наступает, когда найдены все пары

**Управление**

* Выбор сложности: "Легкая"(4* 3, 6 пар), "Средняя"(4* 4, 8 пар), "Сложная"(6*4, 12 пар)
* Кнопка "Новая игра" полностью сбрасывает прогресс и перемешивает карты
```
for (let i = cardsArray.length - 1; i > 0; i--) {
        const j = Math.floor(Math.random() * (i + 1));
        [cardsArray[i], cardsArray[j]] = [cardsArray[j], cardsArray[i]];
    }
```

## FAQ

### Можно ли добавить свои картинки вместо эмодзи?
Да! Замените в файле [cards.json]() массив [cardThemes]() на любые текстовые символы, URL эмодзи или измените код, чтобы вставлять изображения

### Как изменить количество карт или сложность?
В файле [jame.js]() настройте [gridConfig]() :
```
const gridConfig = {
    easy:   { rows: 2, cols: 3, totalPairs: 3 },   // 2x3 = 6 карт (3 пары)
    medium: { rows: 4, cols: 4, totalPairs: 8 },   // 4x4 = 16 карт (8 пар)
    hard:   { rows: 4, cols: 6, totalPairs: 12 }    // 4x6 = 24 карты (12 пар)
};
```

### Можно ли играть на мобильном устройстве?
Да! Проект полностью адаптивен и поддерживает сенсорное управление
